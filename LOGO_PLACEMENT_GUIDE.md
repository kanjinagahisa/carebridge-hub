# ロゴ画像配置ガイド

## 📋 配置手順（1ステップずつ）

### 【ステップ1】アプリ内ロゴ（横長・文字なし）

**配置場所:**
```
📁 public/assets/logo/
```

**ファイル名（正確に入力してください）:**
```
📄 logo-app-light.png
```

**推奨サイズ:**
- 幅: 240px × 高さ: 64px （@2x推奨）
- または 幅: 120px × 高さ: 32px （標準）

**フォーマット:** PNG（透明背景可）

**完全なパス:**
```
public/assets/logo/logo-app-light.png
```

---

### 【ステップ2】PWAアイコン（小）192×192px

**配置場所:**
```

📄 icon-192.png

**サイズ（必須）:** 192px × 192px（正方形）

**フォーマット:** PNG

**完全なパス:**

public/assets/icon/icon-192.png

### 【ステップ4】favicon.ico（ブラウザタブアイコン）

**配置場所:**
```
📁 public/assets/icon/
```

**ファイル名（正確に入力してください）:**
```
📄 favicon.ico
```

**サイズ:** 32×32px または 16×16px（正方形）

**フォーマット:** ICO

**完全なパス:**
```

## ✅ 配置完了後の確認

すべてのファイルを配置したら、以下のコマンドで確認してください：

```bash
# ロゴファイルの確認
ls -lh public/assets/logo/

# アイコンファイルの確認
ls -lh public/assets/icon/
```

すべてのファイルが表示され、ファイルサイズが適切であることを確認してください。

---

## 📝 注意事項

1. **ファイル名は正確に**
   - 大文字小文字を区別します
   - 拡張子（.png、.ico）を正確に入力してください

2. **サイズの確認**
   - アイコンは指定されたサイズ（正方形）で用意してください
   - サイズが異なる場合は、画像編集ソフトでリサイズしてください

3. **フォーマットの確認**
   - PNGファイルは.png拡張子
   - ICOファイルは.ico拡張子

4. **配置後の確認**
   - ファイルが正しい場所に配置されているか確認
   - ファイル名が正確か確認
