# 確認メールテンプレート（日本語）

Supabaseダッシュボードで以下の設定を行ってください：

## 設定手順

1. Supabaseダッシュボード → Authentication → Email Templates
2. 「Confirm signup」テンプレートを選択
3. 以下の内容に置き換えてください：

---

## 件名
```
CareBridge Hub - アカウント登録の確認
```

## 本文（HTML）
```html
<h2>ご登録ありがとうございます</h2>

<p>ご登録を完了いただくために、以下のURLをクリックして、メールアドレスの確認をお願いいたします。</p>

<p><a href="{{ .ConfirmationURL }}">{{ .ConfirmationURL }}</a></p>

<p>※本メールにお心当たりのない場合は、お手数ですが破棄をお願いいたします。</p>

<p>ご不明な点がございましたら、お気軽にお問い合わせください。</p>

<p>CareBridge Hub</p>
```

## 本文（プレーンテキスト）
```
ご登録ありがとうございます

ご登録を完了いただくために、以下のURLをクリックして、メールアドレスの確認をお願いいたします。

{{ .ConfirmationURL }}

※本メールにお心当たりのない場合は、お手数ですが破棄をお願いいたします。

ご不明な点がございましたら、お気軽にお問い合わせください。

CareBridge Hub
```

---

## 注意事項

- `{{ .ConfirmationURL }}` はSupabaseが自動的に置き換える変数です
- HTML版とプレーンテキスト版の両方を設定することを推奨します
